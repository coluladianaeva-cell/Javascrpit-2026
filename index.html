Javascript

Javascript es un lenguaje de programacion dinamico y de alto nivel para crear interfases interactivas y dinamicas

Es uno de los tres pilares fundamentales del desarrollo web.

  • 1HTML:Define la estructura del contenido. Son los elementos que aparecen en la pagina web
  • 2CSS:Define la apariencia, colores, tamaño, estilos, etc
  • 3JS:Define el comportamiento, es decir lo que hace la pagina o sus elementos.

Caracteristicas de Javascript

  1. 👉Lenguaje interpretado: No se necesita compilar.
  2. 👉Basado en objetos: Maneja los elemntos de chtml como objetos: window, date, array, class.
  3. 👉Dinamico: Podemos modificar el contenido en tiempo real (sin recargar)
  4. 👉Versatil y facil de aprender: Aunque nacio especialmente para la web, se usa tambien para servidores, aplicaciones moviles, videojuegos, anmiacion y para IA.
